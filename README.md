## ✨ 주요 기능

### 📝 기본 기능
- 게시글 관리 (CRUD): 게시글 등록, 조회, 수정, 삭제 기능. 모달 창을 활용한 사용자 친화적인 처리 구현
- 목록 처리: 게시글 목록 페이징 처리 기능
- 댓글 관리 (CRUD): 게시글별 댓글 등록, 조회, 수정, 삭제 기능
- 댓글 목록 처리: 댓글 목록 페이징 처리 기능
- 정보 표시: 목록 및 조회 화면에 댓글 수 정보 추가
- UI/UX 개선: 전반적인 화면 디자인 및 사용자 인터페이스 정리

### 🔍 심화 기능 (Advanced Features)
- 검색 기능: 게시글 제목/내용/작성자 등 다양한 조건으로 검색 기능 구현
- 첨부 파일: 게시물 등록/수정 시 첨부 파일 업로드 처리, 등록된 첨부 파일 다운로드 기능 구현
- 인증/인가: 로그인 처리 (Spring Security), 회원가입 설정 및 사용자 정보 관리

### ⚙️ 추가 기능 (Extra Features)
- 마이페이지: 사용자 정보 조회 및 수정, 내가 작성한 게시글/댓글 목록 조회 기능
- 어드민 페이지: 관리자 전용 페이지 구현 (회원 목록 관리, 회원 및 계시글 통계)

## 🛠️ 사용 기술 스택
### Backend 
- 언어: Java
- 프레임워크: Spring
- 의존성 관리: Maven
- 데이터 접근: MyBatis
- 보안 : Spring Security

### Frontend 
- 화면: JSP
- 비동기 처리: AJAX
- 디자인: BootStrap
- 통계 시각화 : Chart.js

### Database 
- DB: 오라클
